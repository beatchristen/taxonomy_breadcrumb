Installing taxonomy_breadcrumb for Drupal 4.7

The taxonomy_breadcrumb module installation instructions are "typical"
for Drupal modules.

1.  Create a folder taxonomy_breadcrumb within your modules folder: 
      modules/taxonomy_breadcrumb

2.  Copy or upload all of the taxonomy_breadcrumb files to this folder.  

3.  Go to the module administration page (admin/modules) while logged in
    as an administrator and enable the module.  The provided .install 
    file will automatically create two database tables when the module
    is first installed.

4. Configure the breadcrumb trail on the module's settings page 
   (admin/settings/taxonomy_breadcrumb).  Configuration instructions 
   are provided on this page.  This module also adds new fields to the
   vocabulary and term add/edit pages (admin/taxonomy).
